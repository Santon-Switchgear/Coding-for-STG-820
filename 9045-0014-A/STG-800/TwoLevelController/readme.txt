TwoLevelController sample code
==============================
External connections/devices:
- Power supply
- Feedback input on AIN1 (Analog Input)
- Output device on DOUT1 (High side output)

Used I/O:
- LED
- Analog input
- Digital output

Function:
- If Voltage at IN1 (V) greater than 5000 mV, the OUT1 will be Hight otherwise Low
- LED toggle every one second
