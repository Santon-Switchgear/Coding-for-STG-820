DMA-20 by CAN sample code
=========================
External connections/devices:
- Power supply
- DMA-20 by CAN

Used I/O:
- LED
- CAN-Transceiver

Function:
- Set up CAN baud-rate to 250kBit
- Show circular the templaqtes 1..3 on the DMA-20 for 15 s with random values
- Show specific template with random values by uncomment main.c:387
- LED toggle every 1000 ms
